Software installation: Node.js & MongoDB JavaScript runtime and non-SQL database
Node.js
Follow instructions for your OS. For Fedora Linux:
sudo dnf install -y gcc-c++ make
curl -sL https://rpm.nodesource.com/setup_13.x | sudo -E bash -
Or whichever version of Node.js you would like to install.
curl -sL https://rpm.nodesource.com/setup_13.x | sudo -E bash -
sudo dnf install nodejs

MongoDB
Follow instructions for your OS. For Fedora Linux, use Redhat installation documentation. You may replace 'yum' with 'dnf'.

Other software installation: Git, VSC & Postman Version control, IDE and API test
Git
Follow instructiosn for your OS. For Fedora Linux:
sudo dnf install -y git
Configure your git user name and user email.
git config --global user.name "YOUR_NAME"
git config --global user.email "YOUR_EMAIL"

Visual Studio Code
Follow instructiosn for your OS or build from source.

Postman
Follow instructiosn for your OS. For Fedora Linux:
sudo dnf install snapd
sudo ln -s /var/lib/snapd/snap /snap
sudo snap install postman
You will need to reboot your computer for application icon to appear. Without reboot, you can launch postman by typing postman in terminal.

Express back end Create express project; install packages
express PROJECT_NAME --no-view --git
Create express project without view engine, but with .gitignore file.
cd PROJECT_NAME
npm i {{x}} (if you plan to upload file)
git init

Edit package.json file. Under 'script', add

"devstart": "DEBUG=sales-report:* nodemon ./bin/www"  (optional. you can use nodemon directly)

Launch server

npm run devstart

ng serve -o
